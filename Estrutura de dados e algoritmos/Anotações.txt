As principais estruturas de dados são:

- Vetores e matrizes: são estruturas simples que podem auxiliar quando há muitas variáveis do mesmo tipo em um algoritmo.
Um vetor é uma variável que armazena várias variáveis do mesmo tipo, começando do ) e vai até infinito.
Um matriz é um vetor que possui duas ou mais dimensções, como linhas e colunas, e possui tamanho fixo.

Exemplo de matriz: 2  1  3  1
                   0  2  5  4
                   3  1  0  6

- Registro: é uma estrutura que fornece um formato especializado para armazenar informações na memória.
Permite armazenar mais de um tipo de dado, por exemplo, dados de um cliente, como CPF, nome, data de cadastro...
Toda estrutura de registro tem um nome e seus campos podem ser acessados por meio do uso do operador ponto (.).

- Lista: as listas podem ser do tipo "lista ligada" ou "lista duplamente ligada".
Na estrutura do tipo lista existem nós. Na lista ligada, os nós conhecem o valor que está sendo armazenado em seu interior e o elemento posterior a ele.
Na lista duplamente ligada, os nós conhecem seu próprio elemento, o elemento anterior e o elemento que vem depois dele.

- Pilha: é uma estrutura de dados que serve como uma coleção de elementos e permite o acesso a somente um item de dados armazenado.
Pode ser do tipo LIFO ou do tipo FIFO.

- Fila: esse tipo de estrutura admite remoção de elementos e inserção de novos, obedendo a regra de que o elemento removido é o mais antigo na estrutura.

- Árvore: esse tipo de estutura oganiza os seus elementos de forma hierarquica. o elemento em seu topo é chamado de raiz, e os elementos subordinados a ela são chamados de nós ou folhas.

- Tabela hash: é uma estrutura de dados especial que associa chaves de pesquisa a valores.
Chave = parte da informação que compõe o elemento a ser manipulado. 

- Grafos: são estruturas que permitem programar a relação entre objetos.
